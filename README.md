מערכת השקיה חכמה

פרויקט השקיה חכמה לניטור והשקיה אוטומטית של צמחים באמצעות חיישנים. המערכת אוספת נתונים על לחות וטמפרטורה, שומרת אותם בבסיס נתונים SQL, ומבצעת השקיה בהתאם לצרכים של כל צמח. הפיתוח מתבצע באמצעות Node.js בצד השרת, React בצד הלקוח לניהול הממשק, ובסיס נתונים SQL לניהול המידע בצורה יעילה. המערכת גמישה ומאפשרת חיבור לחיישנים שונים והתאמות עתידיות.
